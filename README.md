# Weather App

A simple and responsive weather application that allows users to search for any city and view its current weather conditions.

## 🚀 Features

- Search weather by city name
- Real-time temperature
- Humidity
- Wind speed
- Weather condition icons
- Responsive design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API
